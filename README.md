<h1>Быстрый просмотр возможностей приложения</h1>

<p>В корень репозитория помещён <a href="TagGame.jar">исполняемый jar-файл</a> приложения, который можно запустить имея на ПК установленное ПО Java (JRE/JDK). Если приложение не запускается, возможна неполная установка Java (например отсутвие в перменной среды Path значения с папкой bin от Java) или устаревшая версия установленного JRE/JDK.</p>

<h1>Описание приложения</h1>
<p>MVC-приложение</p>
<p>Реализация игры Пятнашки в варианте поля 3*3. Правила стандартные. Поле представляет собой 9 ячеек, 8 из которых заняты числами а девятая - пустое место. </p>
<p>Предусмотрены форматы загрузки и сохранения игры через txt, xml и json файлов. Также предусмотрено случайное создание игры</p>
<p>После запуска необходимо создать игру удобным способом (загрузкой существующей или генерацией случайной). Управление приложением осуществляется средствами строки меню.</p>
<ul>
  <li>
    Загрузка/сохранение игры
  </li>
  <li>
    Создание случайного поля
  </li>
  <li>
    Выключение приложения
  </li>
  <li>

  </li>
</ul>
